# Get Started

# Prerequisites

  • Node.js v14.18.0+open in new window
  • Yarn v1 classicopen in new window

# Manual Installation

  • Step 1: Create and change into a new directory
mkdir dulladmin-starter
cd dulladmin-starter
  • Step 2: Initialize your project
yarn init
  • Step 3: Install dulladmin locally
yarn add -D @dulladmin/cli @dulladmin/generator-arco-vue
PackagePurpose
@dulladmin/clicli toolkit
@dulladmin/generator-arco-vuefrontend code generator with Arco Design Vueopen in new window
  • Step 4: Create client directory
yarn dulladmin client:install

Your project structure is probably like this:

├─ client     # clientDir, all frontend code will be generated here
├─ dulladmin  # dulladminDir, definition files here
│  ├─ resources
│  └─ app.yml
└─ package.json
  • Step 5: Create your first DullAdmin Resource File
vim dulladmin/resources/users.yml
Click to expand sample dulladmin/resources/users.yml
# dulladmin/resources/users.yml
name: 'users'
views:
  index:
    table:
      items:
        - { name: 'id', type: 'int64' }
        - { name: 'name', type: 'string' }
        - { name: 'avatar', type: 'image' }
      sorters:
        - { name: 'id', directions: ['descend', 'ascend'] }
      searchers:
        - { name: 'id', predicate: 'eq' }
        - { name: 'name', predicate: 'cont' }
      pagination:
        per: 5

  show:
    descriptions:
      items:
        - { name: 'id', type: 'int64' }
        - { name: 'name', type: 'string' }
        - { name: 'avatar', type: 'image' }

  new:
    form:
      items:
        - { name: 'name', type: 'string' }
        - { name: 'avatar', type: 'image' }

  edit:
    form:
      items:
        - { name: 'name', type: 'string' }
        - { name: 'avatar', type: 'image' }

  delete:
    form:
      items: []
  • Step 6: Generate client src code
yarn dulladmin build

  • Step 7: Preview
cd client
yarn run-p mock:server dev

It will start a mock server at http://127.0.0.1:3000open in new window to provide backend api service, and a hot-reloading development server at http://127.0.0.1:5173open in new window to serve frontend.

Enter username admin and password 123456 to log in to the system. By now, you should have a basic but functional admin panel. Next, learn about the basics of DullAdmin Resource File.
