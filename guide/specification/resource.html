# Resource

# Definition File

# <Resource @name="users">
name: 'users'
views:
  index:
  new:
  show:
  edit:
  delete:
  mycustom:

that will produce following views:

NameRoute PathPurpose
users#index/usersdisplay a list of all users
users#new/users/newcreate a new user
users#show/users/:iddisplay a specific user
users#edit/users/:id/editedit a specific user
users#delete/users/:id/deletedelete a specific user
users#mycustom/users/:id/mycustomdepends on how you define
