# Value Type

Use [proto3 scalar value types](https://protobuf.dev/programming-guides/proto3/#scalar) to define value type.

| Name     | Proto3 Scalar Value Type | Data Renderer | Form Renderer |
| -------- | ------------------------ | ------------- | ------------- |
| double   | double                   | -             | -             |
| float    | float                    | -             | -             |
| int32    | int32                    | -             | -             |
| int64    | int64                    | -             | -             |
| uint32   | uint32                   | -             | -             |
| uint64   | uint64                   | -             | -             |
| sint32   | sint32                   | -             | -             |
| sint64   | sint64                   | -             | -             |
| fixed32  | fixed32                  | -             | -             |
| fixed64  | fixed64                  | -             | -             |
| sfixed32 | sfixed32                 | -             | -             |
| sfixed64 | sfixed64                 | -             | -             |
| bool     | bool                     | -             | -             |
| string   | string                   | -             | -             |
| datetime | string                   | -             | -             |
| image    | string                   | -             | -             |
| object   | -                        | -             | -             |
